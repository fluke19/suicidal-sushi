Portfolio

Eteach mobile

Eteach had a mobile site but it was outdated and in dire need of a refresh. The focus was allowing the users to search for jobs as easy as possible because search is the main focus being a job board.

I came up with a design that was simplistic and added more emphasis on the search. To have the search as the main focus I introduced a basic search on the home page and a search that can be accessed anywhere from the navigation. The site is now live but should be looked at on your mobile device.

For Eteach.com

Eteach Recruiter

Eteach first and foremost are a teaching recruitment company, and it was decided that to do the recruitment job more efficiently they needed a recruitment system.

This was more of a detailed design because of the amount of information being displayed on a single page. I worked closely with the user experience manager in making sure the that the user could easily navigate through the journey to allow users to search and create assignments for different teachers whilst making sure the UI was easy to understand.

For Eteach.com

Eteach International

Not too long ago when Eteach was expanding they decided to launch Eteach International, this would act as an extended site to Eteach. The code was going to be based off of the Eteach site but give it a more modern feel.

We wanted teachers landing on the homepage to be enticed to go and teach aboard, so we thought having a large image slider and interactive world map would help whilst showing available jobs in different countries. Recently I have done work improving the page load and fine tuning, this will be going live in the next few weeks.

For Eteach.com
